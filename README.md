#Messenger Real Time Chat App

Swift project to create a real time chat application in Swift 5 using Firebase.

## Features
- Facebook Log In
- Google Sign In
- Email/Pass Registration/Log In
- Photo Messages
- Video messages
- Real Time Conversations
- Location messages
- Search for Users
- Deleting Conversations
- User Profile
- Dark Mode Support
